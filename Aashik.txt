To install Sleuth Kit (TSK) on Kali Linux, you can follow these steps. Kali Linux repositories often include Sleuth Kit, so installation is usually straightforward.

âœ… Method 1: Install via APT (Recommended)


This is the easiest and safest method.


1. Update your package list:


sudo apt update




2. Install Sleuth Kit:


sudo apt install sleuthkit




3. Verify installation:


fls -V


You should see version information if the tool is installed correctly.

âœ… Method 2: Build from Source (Optional - for latest version)


If you want the latest version, you can build it from the source:


1. Install dependencies:


sudo apt update
sudo apt install build-essential autoconf libtool libafflib-dev libewf-dev zlib1g-dev libsqlite3-dev




2. Download the latest source code: Go to https://github.com/sleuthkit/sleuthkit and download the latest release:


git clone https://github.com/sleuthkit/sleuthkit.git
cd sleuthkit




3. Build and install:


./bootstrap
./configure
make
sudo make install




4. Check installation:


fls -V
